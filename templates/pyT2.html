2. Strings

2.1 Basics

A string is a sequence of characters, a character being any letter, number or other symbol. In Python, you represent strings with quotation marks. For example:

'Hello World' is a string

Hello is not a string (because it doesn't have quotation marks around it)

You can use double quotations as well. For example:

"Hello World" is a string

There is no difference between single and double quotations in python. You can verify if something is a string using the type function:

>>> type('Hello World')
<class 'str'>

'str' simply meaning that the content in the brackets is a string.

2.2 What are methods?

Methods are procedures you can perform on strings (and other data types that will be covered later like lists). Python has a set of built-in methods that you can use on strings and these can change various features of the string.

2.3 String Methods

Firstly, let's say you have the string: 'Hello World'

If you want to capitalise the whole string, you could use upper which is a method giving you a string with all the letters capitalised.

>>> 'Hello World'.upper()
HELLO WORLD

Note that if you try to capitalise a string which is already in caps, upper will return that string:

>>> 'HELLO WORLD'.upper()
HELLO WORLD

Conversely, if you wanted to convert a string completely to lowercase, you can use the lower function:

>>> 'Hello World'.lower()
'hello world'

Additionally, there are methods to find out properties of a string. For example, you can use the isupper method which returns either True or False (we will cover many uses of True and False values later on). If a string only has caps, isupper will output True, otherwise it will output False:

>>>'HELLO WORLD'.isupper()
True

>>>'Hello World'.isupper()
False

The last method we will cover in this article is one to count how many times a string occurs in another string. Let's say we wanted to see how many times 'Hello' occurs in 'Hello World'

>>> 'Hello World'.count('Hello')
1

You can also use it to count how many times a certain letter occurs:

>>> 'Hello World'.count('o')
2

2.4 Concatenation

The process of joining strings is called concatenation. In python we can use + for this purpose. For example:

>>> 'Hello' + 'World'
'HelloWorld'

Notice how there is no space between the two words. You need to consider this very often and there is a very simple fix: adding a space at the end of the first string.

>>> 'Hello ' + 'World'
Hello World
